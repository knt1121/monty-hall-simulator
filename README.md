# 🎲 Monty Hall Problem Simulator

モンティホール問題を実際に「動かして」確かめるためのシンプルなWebシミュレーターです。

「扉を変えると本当に勝率は上がるのか？」  
その都市伝説を、確率ではなく体験で検証できます。

---

## 🧠 モンティホール問題とは

3つの扉のうち1つに当たりがあります。

1. プレイヤーが1つ選ぶ  
2. 司会者がハズレの扉を1つ開ける  
3. その後、選択を変えるかどうか決める  

このとき――  
**選択を変えた方が勝率は約2/3になる**と言われています。

本当にそうなるのか、実演します。

---

## ✨ 機能

- シミュレーションで大量試行
- 扉を変更した場合の勝率・負け率を表示
- シンプルなUIで直感操作

---

## 🚀 使い方

1. 扉を1つ選ぶ  
2. 開示後に変更するか決める  
3. 結果を確認  
4. シミュレーションで確率を観察
